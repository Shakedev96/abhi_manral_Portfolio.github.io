Projects

Pop Duel

Project Description : This is a PvP Local Multiplayer game, where max 4 players can join in a battle royale where the last one to survive is the winner. This was game was made with the collaboration of 6 people for the Global Game Jam, where the Theme given to us was "Bubbles".

My Role: Made an animation controller to handle various Movements, Jump, Dash, L/R Strafing. Made the bullets (Bubbles) material using shader graphs and used vfx graphs for the trail and impact effects.

Gas The Bully

Project Description : This is a working prototype of a mobile game, developed during a GameJam, for the Theme "Evolution". The mobile(For Android) game has 3 levels, where the camera angle changes from platformer to perspective to ThirdPerson, showcasing the evoultion of games through Changing Camera View and mechanics.

In the game, level 1: Lunch Bell - while avoiding falling on the ground, and the bullies, the player must Fart(Jump) from table to table to make it to the Exit. Player movement is side-scrolling platformer based i.e Left/ Right movement and Jump. Level 2: Corridor - the user can now move in 3D space in this isometric view level. while carefully avoiding the bullies, user needs to collect 30 Bugers to finish the game.

My Role: As the Dev lead, i was responsible for programming the core mechanics for the game, taking inspiration from jetpack joyride, implemented similar jump mechanics.

ToyBattleSim

Project Description : A 3D toy fighter plane simulator game (Currently in Prototype stage) with fast-paced arcade-style gameplay where players control a toy plane. The primary objective is to skillfully pilot the plane, avoiding relentless attacks of AI-controlled anti-plane guns that are determined to shoot you down. Survive as long as possible while collecting power-ups and achieving high scores.

My Role: Implemented basic Flight Mechanics for a plane, accurate physics based movement to be added. Made AI guns that target and track the player and shoot at the player, used Finite State Machine to achieve switch between states.
